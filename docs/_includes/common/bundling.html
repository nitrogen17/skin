Bundling

Skin supports Webpack and Lasso.js.

Webpack

Add @ebay/skin, @ebay/browserslist-config as dependencies and less, less-loader as dev dependencies to package.json.

{% highlight json %} { "dependencies": { "@ebay/browserslist-config": "^1", "@ebay/skin": "^{{ page.version | slice: 0, 2 }}", }, "devDependencies": { "less": "^3", "less-loader": "^5" } } {% endhighlight %}

Now in your webpack configuration do:

{% highlight js %} module.exports = { module: { rules: [ { test: /\.less$/, loader: 'less-loader', // compiles Less to CSS }, ], }, }; {% endhighlight %}

Usage

{% highlight js %} import "@ebay/skin/button.less"; {% endhighlight %}

You can also leverage webpack's webpack-resolver:

{% highlight js %} import "~@ebay/skin/button.less"; {% endhighlight %}

Lasso.js

Add the @ebay/skin module as a browser.json dependency to include the entire set of Skin modules.

{% highlight json %} { "dependencies": [ "@ebay/skin" ] } {% endhighlight %}

Or, alternatively, by specifying modules on a per-need basis.

{% highlight json %} { "dependencies": [ "@ebay/skin/less", "@ebay/skin/global", "@ebay/skin/button", "@ebay/skin/icon" ] } {% endhighlight %}
